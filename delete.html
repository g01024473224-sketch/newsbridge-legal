주식회사 웨일즈컴퍼니는 「개인정보 보호법」 및 Google Play / App Store 정책에 따라 이용자가 본인의 데이터에 대한 열람·정정·삭제를 언제든 요청할 수 있도록 절차를 제공합니다. 본 페이지는 뉴스브릿지(번들 ID: com.whalescompany.newsbridge) 의 데이터 삭제 요청 안내입니다.

1. 삭제 대상 데이터

항목설명
스크랩한 기사 /userScraps/{사용자ID} 경로에 저장된 본인의 스크랩 목록 및 기사 스냅샷 전체
계정 정보 Firebase Authentication 에 저장된 계정 정보 — 연동된 Google/Apple 이메일, 인증 식별자 포함
익명 사용자 ID 회원가입 없이 익명으로 이용한 경우 발급된 익명 UID 및 그에 연결된 모든 사용자별 데이터
키워드 알림 설정 /userTokens/{사용자ID} 에 저장된 FCM 푸시 토큰 및 등록 키워드
알림 발송 기록 /sentAlerts/{사용자ID} — 중복 발송 방지를 위해 보관되는 기록
편향 분석 사용 카운트 /users/{사용자ID}/biasAnalysisCount — 일일 무료 한도 관리용

2. 삭제 요청 방법

이메일 요청

요청 메일에 포함해 주실 내용

  1. 식별 정보 — 연동된 Google/Apple 이메일 주소 또는 익명 UID
    익명 UID 를 모르시면 가입에 사용한 이메일만 적어주셔도 처리 가능합니다.
  2. 삭제 범위
    • 전체 삭제 (계정 + 모든 관련 데이터)
    • 스크랩 기사만 삭제
    • 키워드 알림 설정만 삭제
    • 기타 (자유 기재)
  3. 본인 확인 — 동일 이메일 주소에서 발송하실 경우 추가 절차는 통상 생략됩니다.

3. 처리 기간

  • 요청 접수 후 영업일 기준 7일 이내에 처리 결과를 회신해 드립니다.
  • 처리 완료 후 회사가 보유하던 해당 데이터는 즉시 영구 삭제됩니다.
  • 법령에 따라 일정 기간 보존이 요구되는 정보(예: 접속 로그 3개월)는 해당 기간 경과 후 자동 파기됩니다.

4. 앱에서 직접 삭제 (선택)

다음 항목은 앱 내에서 즉시 삭제할 수 있습니다.

  • 스크랩 해제 — 기사 카드의 북마크 아이콘을 다시 탭하면 해당 스크랩이 즉시 삭제됩니다.
  • 키워드 삭제 — 설정 → 키워드 알림 → 등록된 키워드 칩을 탭하면 즉시 삭제됩니다.
  • 광고 식별자 재설정 — 단말 OS 설정에서 처리 (iOS: 개인정보 보호 → 추적 / Android: 설정 → 광고).

5. 삭제 후 영향

데이터를 삭제하시면 다른 기기에서 같은 계정으로 로그인해도 스크랩 / 설정이 복원되지 않습니다. 필요한 데이터는 삭제 전 PDF 저장(프리미엄) 또는 캡처로 보관해 두시기를 권장합니다.
